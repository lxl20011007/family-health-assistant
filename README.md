# 家庭健康助手

一个功能完整的家庭健康管理应用，支持实时云同步。

## 🎯 功能特性

### 📱 核心功能
- **家庭成员管理** - 添加、编辑、删除家庭成员
- **健康指标记录** - 血压、血糖、心率、身高、体重、BMI
- **饮食管理** - 记录每日饮食，自动计算营养信息
- **运动记录** - 记录运动类型、时长、消耗热量
- **用药提醒** - 设置用药时间和提醒

### ☁️ 云同步功能
- **实时同步** - 添加数据后立即同步到云端
- **自动同步** - 每 30 秒自动检查一次云端更新
- **多设备共享** - 家人可以在不同设备上看到同一份数据
- **离线支持** - 离线时数据保存在本地，网络恢复后自动同步

### 🎨 用户体验
- **响应式设计** - 支持手机、平板、电脑
- **简洁界面** - 易用的操作界面
- **数据可视化** - 清晰的统计和趋势展示
- **BMI 自动计算** - 输入身高体重自动计算 BMI

## 📊 数据存储

### 本地存储
- 所有数据默认保存在浏览器的 localStorage 中
- 数据不会上传到任何服务器

### 云同步（可选）
- 支持连接 Supabase 进行云同步
- 需要自己的 Supabase 项目
- 完全可选，不影响本地使用

## 🔐 隐私保护

- ✅ 所有数据默认保存在本地
- ✅ 云同步完全可选
- ✅ 不收集任何个人信息
- ✅ 支持一键清空所有数据

## 📝 使用指南

### 配置云同步
1. 点击右上角 ☁️ 云同步按钮
2. 输入你的 Supabase URL 和 API Key
3. 点击"保存并启用"
4. 所有本地数据会自动上传到云端

### 清空所有数据
1. 点击右上角 🗑️ 垃圾桶图标
2. 勾选确认框
3. 点击"确认删除所有数据"
