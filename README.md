# 家庭健康助手

一个本地运行的家庭健康管理工具，所有数据都保存在您的设备中，保护您的隐私。

## 🎯 功能特点

### 1. 家庭成员管理
- 添加/编辑/删除家庭成员
- 记录基本信息：姓名、性别、出生日期、备注
- 支持多成员管理

### 2. 健康指标记录
- **血压**：记录收缩压和舒张压
- **血糖**：记录血糖值（mmol/L）
- **心率**：记录心率（次/分）
- 按时间倒序展示记录
- 支持按类型筛选

### 3. 用药提醒
- 设置药品名称和用法用量
- 自定义提醒时间（可设置多个时间点）
- 启用/停用提醒功能
- 页面内弹窗提醒（不依赖系统通知）

### 4. 运动记录
- 记录运动类型（跑步、游泳、健身等）
- 记录运动时长（分钟）
- 记录运动日期
- 历史记录查看

## 🚀 快速开始

### 方法一：直接打开（推荐）
1. 双击打开 `index.html` 文件
2. 应用将在您的默认浏览器中打开
3. 所有数据自动保存在浏览器的 localStorage 中

### 方法二：使用本地服务器
如果您遇到跨域问题，可以使用以下方法：

#### Python 3
```bash
# 在应用目录下运行
python -m http.server 8000
# 然后在浏览器中访问 http://localhost:8000
```

#### Node.js
```bash
# 安装 http-server
npm install -g http-server

# 在应用目录下运行
http-server
# 然后在浏览器中访问 http://localhost:8080
```

## 📱 使用指南

### 第一步：添加家庭成员
1. 点击"添加成员"按钮
2. 填写姓名、性别、出生日期等信息
3. 点击"添加"按钮保存

### 第二步：记录健康指标
1. 从顶部选择家庭成员
2. 切换到"健康指标"标签页
3. 点击"添加记录"按钮
4. 选择记录类型并填写数值
5. 点击"添加记录"保存

### 第三步：设置用药提醒
1. 确保已选择家庭成员
2. 切换到"用药提醒"标签页
3. 点击"添加提醒"按钮
4. 填写药品信息、用法用量和提醒时间
5. 点击"添加提醒"保存

### 第四步：记录运动
1. 确保已选择家庭成员
2. 切换到"运动记录"标签页
3. 点击"添加记录"按钮
4. 填写运动类型、时长和日期
5. 点击"添加记录"保存

## 🔒 数据安全

### 数据存储
- 所有数据都保存在浏览器的 **localStorage** 中
- 数据不会上传到任何服务器
- 数据只在您的设备上存在

### 数据备份
由于数据存储在本地，建议定期备份：
1. 打开浏览器的开发者工具（F12）
2. 切换到"Application"或"存储"标签页
3. 找到"Local Storage"部分
4. 导出相关数据

### 数据恢复
如果需要恢复数据，可以将备份的数据重新导入到 localStorage 中。

1. **浏览器兼容性**：建议使用 Chrome、Firefox、Edge 等现代浏览器
2. **数据持久性**：清除浏览器缓存会删除所有数据
3. **隐私保护**：不要在公共电脑上使用，或使用后清除数据
4. **用药提醒**：页面内提醒，需要保持页面打开
