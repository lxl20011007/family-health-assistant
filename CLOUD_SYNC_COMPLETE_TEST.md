## 🧪 完整测试流程

### **测试环境准备**

1. **打开两个浏览器标签页**
   - 标签页 A：你的手机（或浏览器）
   - 标签页 B：家人的手机（或另一个浏览器）

2. **两个标签页都配置 Supabase**
   - 打开云同步设置
   - 输入相同的 URL 和 API Key
   - 点击"保存并启用"
   - 确保显示"已连接"

3. **打开浏览器控制台**
   - 按 F12 打开开发者工具
   - 切换到"控制台"标签
   - 这样可以看到同步日志

---

## 🔍 故障排查

### **如果看不到数据**

1. **检查浏览器控制台**
   - 按 F12 打开开发者工具
   - 查看是否有错误信息
   - 记下错误信息

2. **检查 Supabase 连接**
   - 确保两个设备使用相同的 URL 和 API Key
   - 点击"测试连接"确保连接成功

3. **检查 Supabase 数据库**
   - 登录 Supabase 控制面板
   - 查看相应的表是否有数据
   - 运行 SQL 查询验证

4. **手动刷新**
   - 按 F5 或 Ctrl+R 刷新页面
   - 或者关闭应用重新打开

### **如果同步失败**

1. **查看控制台错误**
   - 应该看到 `❌ 同步失败` 的错误信息
   - 记下具体的错误内容

2. **检查网络连接**
   - 确保两个设备都有网络连接
   - 尝试访问其他网站确认网络正常

3. **检查 Supabase 配置**
   - URL 是否正确
   - API Key 是否正确
   - 是否有拼写错误
