## 🚀 快速开始

### **第一步：配置 Supabase**
1. 点击右上角 "云同步设置" 按钮
2. 输入你的 Supabase URL 和 API Key
3. 点击 "测试连接" 验证
4. 点击 "保存并启用"

### **第二步：注册账号**
1. 点击 "登录" 按钮
2. 切换到 "注册" 标签
3. 输入邮箱和密码（至少6位）
4. 点击 "注册"

### **第三步：开始使用**
1. 使用注册的账号登录
2. 系统会提示导入本地数据（可选）
3. 开始添加家庭成员和健康记录
4. 在其他设备上用同一账号登录，数据自动同步

## 📱 多设备使用

### **同一家庭**
- 所有设备使用 **同一账号** 登录

### **数据安全**
- 每个账号数据完全隔离

## 🔧 常见问题

### **Q: 如何获取 Supabase 配置？**
A: 
1. 访问 https://supabase.com 注册
2. 创建新项目
3. 在项目设置 → API 中获取 URL 和 anon key

### **Q: 本地数据会丢失吗？**
A: 不会！首次登录时会提示导入本地数据到云端。

### **Q: 支持离线使用吗？**
A: 支持！网络断开时使用本地缓存，恢复后自动同步。

### **Q: 如何清空数据？**
A: 点击右上角垃圾桶图标，勾选确认后清空。
