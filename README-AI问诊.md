# AI问诊模块使用说明

## 功能概述
AI问诊模块为家庭健康助手应用添加了智能健康咨询功能，可以调用DeepSeek大模型进行实时对话，了解健康状况并提供建议。

## 主要功能

### 1. API密钥配置
- 支持DeepSeek API密钥的本地存储
- 密钥仅保存在浏览器localStorage中，不会上传到服务器
- 支持密钥的保存、更新和清除

### 2. 智能对话
- 实时对话了解健康状况
- 支持上下文理解（最近10条对话历史）
- 自动整合家庭成员健康记录作为对话背景
- 支持Markdown格式的消息显示

### 3. 对话管理
- 开始新对话
- 清空当前对话
- 自动保存对话历史（最多50条）

### 4. 健康背景整合
- 自动获取当前选择的家庭成员信息
- 整合最近的健康记录（血压、血糖、心率等）
- 为AI提供更准确的健康背景信息

## 使用步骤

### 第一步：获取API密钥
1. 访问 [DeepSeek平台](https://platform.deepseek.com/api_keys)
2. 注册/登录账户
3. 创建新的API密钥
4. 复制生成的密钥

### 第二步：配置API密钥
1. 在应用中选择"AI问诊"标签页
2. 在API密钥配置区域输入您的DeepSeek API密钥
3. 点击"保存密钥"按钮
4. 密钥将安全地保存在您的浏览器中

### 第三步：开始对话
1. 选择要咨询的家庭成员（可选）
2. 在聊天输入框中输入您的健康问题
3. 点击"发送"按钮或按Enter键
4. AI助手将根据您的健康记录和问题提供专业建议

## 对话示例

### 示例1：症状咨询
**用户**：我今天感觉头晕，有点乏力，应该怎么办？

**AI助手**：头晕和乏力可能由多种原因引起，包括：
- 低血压或低血糖
- 贫血
- 脱水
- 睡眠不足
- 压力过大

建议您：
1. 先测量血压和血糖
2. 确保充足饮水
3. 适当休息
4. 如果症状持续或加重，请及时就医

### 示例2：生活方式建议
**用户**：我想改善睡眠质量，有什么建议？

**AI助手**：改善睡眠质量的建议：
1. **规律作息**：每天固定时间睡觉和起床
2. **睡前放松**：睡前1小时避免使用电子设备
3. **环境优化**：保持卧室安静、黑暗、凉爽
4. **饮食注意**：避免睡前摄入咖啡因和大量食物
5. **适度运动**：白天适当运动，但避免睡前剧烈运动

## 常见问题

### Q1: API密钥安全吗？
A: 是的，API密钥仅保存在您的浏览器localStorage中，不会上传到任何服务器。

### Q2: 对话历史会保存多久？
A: 对话历史会一直保存在您的浏览器中，直到您手动清除或使用浏览器的清除数据功能。

### Q3: 支持哪些健康问题咨询？
A: 支持一般健康咨询、症状分析、生活方式建议、用药注意事项等，但不支持诊断或开具处方。

### Q4: 如何获取更好的回答？
A: 建议：
1. 选择家庭成员，让AI了解健康背景
2. 描述具体症状和持续时间
3. 提供相关健康指标数据
4. 保持对话连贯性
