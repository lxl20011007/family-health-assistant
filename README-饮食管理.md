# 饮食管理模块使用说明

## 🍽️ 功能概述
已将原来的"用药提醒"模块成功改为"饮食管理"模块，用于记录家庭成员每日三餐的饮食状况，自动计算营养成分。

## ✨ 主要功能

### 1. 饮食记录
- 记录每日三餐（早餐、午餐、晚餐、加餐）
- 支持多种食物选择（内置100+种常见食物）
- 支持自定义食物名称
- 多种计量单位（g、ml、个、碗、杯等）
- 自动记录时间

### 2. 营养计算
- **自动计算**：碳水、蛋白质、脂肪、膳食纤维、热量
- **实时预览**：添加记录时实时显示营养估算
- **每日汇总**：按餐次统计营养摄入
- **历史记录**：保存所有饮食记录

### 3. 数据管理
- 按日期筛选饮食记录
- 按家庭成员分类
- 支持删除记录
- 本地存储，保护隐私

## 📋 使用步骤

### 第一步：选择家庭成员
1. 在顶部选择框中选择家庭成员
2. 如果没有成员，点击"添加成员"按钮创建

### 第二步：添加饮食记录
1. 点击左侧导航栏的"饮食管理"
2. 点击"添加记录"按钮
3. 填写饮食信息：
   - 选择餐次（早餐/午餐/晚餐/加餐）
   - 选择或输入食物名称
   - 输入数量和单位
   - （可选）添加备注
4. 实时查看营养估算
5. 点击"保存记录"

### 第三步：查看营养汇总
- 系统自动计算每日营养摄入
- 显示总热量、碳水、蛋白质、脂肪、膳食纤维
- 按餐次分布显示营养摄入

## 🍎 食物数据库
内置常见食物营养成分数据库，包含：
- **主食类**：米饭、面条、馒头、面包等
- **蛋白质类**：鸡蛋、鸡肉、猪肉、牛肉、鱼肉、豆腐等
- **蔬菜类**：青菜、西兰花、胡萝卜、番茄、黄瓜等
- **水果类**：苹果、香蕉、橙子、西瓜、葡萄等
- **油脂类**：食用油、坚果、花生、核桃等
- **饮料类**：水、茶、咖啡、果汁等

## 🔧 技术特点

### 1. 智能计算
- 根据食物类型和数量自动计算营养成分
- 支持不同单位的智能转换
- 实时更新营养预览

### 2. 数据安全
- 所有数据存储在本地浏览器中
- 不会上传到任何服务器
- 支持数据导出/导入

### 3. 用户体验
- 响应式设计，支持手机和电脑
- 简洁直观的界面
- 实时反馈和提示

## 📱 移动端适配
- 自动适应手机屏幕大小
- 触摸友好的按钮和输入框
- 优化的小屏幕布局

## 🔄 数据备份
建议定期备份数据：
1. 导出数据：在浏览器开发者工具中导出localStorage数据
2. 导入数据：将备份数据导入到新设备

## ⚠️ 注意事项
1. 营养计算基于标准食物数据库，实际值可能略有差异
2. 自定义食物的营养计算使用平均值估算
3. 建议结合专业营养师建议使用
4. 数据存储在本地，更换设备或清除浏览器数据会丢失

## 🚀 快速开始
1. 打开 `index.html` 文件
2. 添加家庭成员
3. 开始记录饮食
4. 查看营养分析
